## Use this extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/denkopro100/kljucavnica-na-cas-** and import

## Edit this extension ![Build status badge](https://github.com/denkopro100/kljucavnica-na-cas-/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/denkopro100/kljucavnica-na-cas-** and click import
